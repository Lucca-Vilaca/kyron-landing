# kyron-landing

Landing page estática do projeto Kyron, criada fora da pasta principal do app Flutter.

## Estrutura
- `index.html`: estrutura da página
- `styles.css`: identidade visual e responsividade
- `script.js`: animações simples de reveal e ano do rodapé
- `assets/kyron_logo.jpeg`: logo reutilizada do app

## Como abrir
Basta abrir `index.html` no navegador.

Se quiser servir localmente:

```bash
cd /Users/luccavilaca/Desktop/Kyron/kyron-landing
python3 -m http.server 8080
```

Depois acesse `http://localhost:8080`.
